# 🎨 색의 진하기와 비율 체험 (초등 수학)

이 프로젝트는 초등학생이 비율을 이용해 색을 섞어 색의 진하기를 이해하도록 돕는 간단한 웹 앱(Streamlit)입니다.

- 지금 버전: 한 가지 색을 선택하고 흰색과의 비율을 조절하여 색의 진하기를 관찰할 수 있습니다.
- 그라데이션 팔레트(선택 색 → 흰색)와 붓 마커(🖌️)로 현재 농도를 직관적으로 확인할 수 있습니다.
- 그라데이션 색 판(색 A → 색 B)을 표시하며, 비율을 바꾸면 현재 섞이는 농도에 해당하는 위치에 마커가 표시됩니다.
- 퍼센트 슬라이더와 정수 비율 입력이 동기화되어 동적인 학습이 가능합니다.
- 퍼센트 슬라이더와 정수 비율 입력이 동기화되어 동적인 학습이 가능합니다.
- 채널별(R/G/B) 가중치를 따로 조절하여 각 채널별 섞임 정도를 관찰할 수 있습니다. 이때 각 채널별 위치는 그라데이션에서 색깔(빨강/초록/파랑) 표시로 보여집니다.
- 비율을 분수 및 백분율로 계산해서 보여줍니다.
- RGB 채널별 가중 평균 계산을 통해 실제로 섞인 색의 숫자값(HEX, RGB)을 제공합니다.
- 간단한 연습문제와 설명이 포함되어 있어 수업 및 가정 학습에 쓸 수 있습니다.

### 사용 방법
1. 의존성 설치

```bash
pip install -r requirements.txt
```

2. 앱 실행

```bash
streamlit run streamlit_app.py
```

3. 웹 브라우저에서 제공되는 로컬 주소(예: http://localhost:8501)에 접속하면 앱을 사용할 수 있습니다.

### 이번 버전 주요 기능
- 한 가지 색을 선택하여 흰색과의 비율을 입력(정수)합니다.
- 비율에 따라 팔레트에서 붓 이모티콘(🖌️)으로 현재 농도 위치를 표시합니다. (참고: 팔레트는 왼쪽이 선택한 색이 진한 쪽, 오른쪽이 흰색입니다. 붓이 왼쪽에 가까울수록 선택 색이 더 큽니다.)
- HEX/RGB 결과값과 최종 혼합 색을 화면에서 확인할 수 있습니다.
- 그림판 기능: 혼합한 색으로 직접 그림을 그릴 수 있으며, 그린 이미지를 PNG로 다운로드할 수 있습니다. (추가 패키지가 필요합니다)

### 학습 목표
- 비율(예: 1:3)을 분수와 백분율로 해석할 수 있다.
- 비율을 이용해 두 색을 섞는 방법을 이해한다.
- 간단한 가중 평균 개념(각 항목에 대한 비중을 더하고 나누는 계산)을 익힌다.

### 채널별 섞기 사용법
- '채널별 가중치 적용'을 체크하면 R/G/B 채널을 개별적으로 0~100%로 조절할 수 있습니다.
- 각 채널의 퍼센트는 `율을 의미합니다 (예: R 채널 80%라면 A의 R값이 80% 비중을 색 A`가 해당 채널에 기여하는 비갖습니다).
- 채널별 퍼센트는 평균값이 그라데이션의 주요 마커로 표시되며, 각 채널별 마커(빨강/초록/파랑)도 그라데이션에 따로 보입니다.

### 교사용 팁
- 수업 중 예시로 색을 사용하여 학생의 흥미를 이끌 수 있습니다.
- 학생들에게 문제를 내고 직접 비율을 입력하게 해서 결과를 관찰하도록 합니다.
